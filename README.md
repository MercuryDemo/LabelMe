## 运行说明

1. 下载GitHub仓库下ReactLabel文件夹
2. cd frontend 运行`npm install`安装依赖文件
3. 导入label.sql数据库建表文件
5. 运行`npm start`运行前端
6. 修改app.py中连接本地数据库语句，修改为本地数据库用户名、密码
7. 运行app.py后端文件
8. 请保证你的设备安装了MySQL，和opencv-python依赖（`pip install opencv-python`），以保证后端成功运行
